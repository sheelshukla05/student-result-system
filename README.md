# Student Result Management System

This is a beginner-friendly Python project that calculates student results based on marks entered by the user.

## Features
- Takes student marks as input
- Calculates total and percentage
- Generates grades automatically
- Displays pass/fail status

## Technologies Used
- Python

## File
- main.py → Main Python program

## Output
The program displays the student's total marks, percentage, grade, and result status.
